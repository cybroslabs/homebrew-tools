## openxml-embedded-extractor

Extract embedded files from OpenXML documents (docx, xlsx, pptx)

### Synopsis

openxml-embedded-extractor is a command-line tool to extract embedded files
from Microsoft Office documents (Word, Excel, PowerPoint).

It supports:
- Automatic file type detection using magic bytes
- OLE (Compound File) container extraction
- Multiple file formats (PDF, Office docs, images, media files, text files)
- Original filename preservation when available
- Batch processing with glob patterns

```
openxml-embedded-extractor [files...] [flags]
```

### Examples

```
  openxml-embedded-extractor document.docx
  openxml-embedded-extractor file1.docx file2.xlsx file3.pptx
  openxml-embedded-extractor *.docx
  openxml-embedded-extractor ./documents/*.xlsx
```
