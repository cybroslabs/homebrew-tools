## ouroctl apply gulf

Apply vendor-delivered crypto material to devices

### Synopsis

Upload a vendor delivery file (e.g. GULF) containing device crypto material for import; the vendor format is auto-detected unless --format is given.

```
ouroctl apply gulf [flags]
```

### Examples

```
  # Apply crypto material with decryption secret (auto-detected format)
  ouroctl apply gulf --file device_secrets.xml --decryption-secret-id secret_id

  # Apply a plaintext ISKRA delivery file (no decryption secret needed)
  ouroctl apply gulf --file device_secrets.xml --format iskra
```

### Options

```
  -s, --decryption-secret-id string   ID of the data decryption secret to use
  -f, --file string                   Path to the vendor crypto material file to apply
      --format string                 Vendor format: auto (default), iskra, lg, zpa
  -h, --help                          help for gulf
      --write-mode string             create-only (default) or create-or-update
```
