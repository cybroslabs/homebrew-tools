## ouroctl apply template

Apply Excel template to create or update objects

### Synopsis

Apply an Excel template to create or update devices, communication units, registers, variables, device configuration templates, device groups, or custom fields.

```
ouroctl apply template [flags]
```

### Examples

```
  # Only create new objects, do not replace existing ones
  ouroctl apply template --file all_in_one.xlsx

  # Create or replace existing objects
  ouroctl apply template --file all_in_one.xlsx --replace
  ouroctl apply template --file all_in_one.xlsx --replace --setlink
```

### Options

```
  -f, --file string   Path to the template file to apply
  -h, --help          help for template
  -n, --no-cache      Disable caching and fetch all data from server
  -r, --replace       Replace existing objects (default: create new only)
  -l, --setlink       Enable set or replace links between device and comunication unit
```
