## ouroctl get delete-template

Retrieve a template for deleting objects

### Synopsis

Retrieves a template for deleting objects that can be used to cleanup objects in OuroPlatform.
Object is specified by object name and type.
The template is provided as an Excel file.

```
ouroctl get delete-template [flags]
```

  ouroctl get delete-template --object-type DEVICE_CONFIGURATION_TEMPLATE
  ouroctl get delete-template --object-type DEVICE_CONFIGURATION_REGISTER
  ouroctl get delete-template --object-type VARIABLE
